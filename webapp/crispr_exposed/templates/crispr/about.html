What is CRISPR Exposed?

CRISPR Exposed is a database of CRISPR elements. It is akin to other projects such as CRISPRdb and CRISPI but uses a larger dataset for the construction of the database.

The database contains CRISPR arrays and spacers of all possible bacterial/archea genomes available from the NCBI genome database. On the 1st of December 2015, that represents about 52000 genomes. All the genomes were processed using CRT, which is a computational tool that helps detect CRISPR arrays in genomes.
